# Skills

This directory is for documenting specialized skills, workflows, and patterns that AI assistants should use when working in this repository.

## Purpose

- Capture domain knowledge about VS Code extensions and TypeScript patterns.
- Provide guidance on how to handle build, packaging, and extension activation logic.
- Store reusable guidance for agents and copilots.

## Suggested structure

- `README.md`: high-level skill guidance.
- Additional files for large skills or patterns.

## Notes

Keep skill files focused on actionable practices and repository-specific design decisions.
