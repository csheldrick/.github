# Agents

This directory is reserved for AI agent definitions, automation helpers, and repository-specific reasoning artifacts.

## Purpose

- Store agent instructions and workflows used for editing or maintaining this repository.
- Document how AI assistants should approach feature work, bug fixes, and refactors.

## Suggested content

- Custom agent configuration files
- Behavior guidelines for repository automation
- Reference links for build and packaging tasks

## Notes

This directory is intentionally lightweight. Add files only when a persistent agent behavior or automation workflow needs to be documented.
